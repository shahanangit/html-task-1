# Frontend Web Development Projects

## Overview
This repository contains frontend web development projects built using HTML, CSS, and JavaScript.

## Projects

### Microsoft Homepage Clone
A responsive clone of the Microsoft homepage featuring navigation menus, product sections, business sections, and footer components.

### Swiggy Clone
A food delivery website clone inspired by Swiggy, developed to practice frontend layout design, responsive interfaces, and user experience implementation.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Font Awesome

## Purpose
These projects were developed to strengthen frontend development skills, responsive web design, and UI implementation.
